## Run Locally

Clone the project

```bash
  git clone https://github.com/gutiermoran/gutiermoran.com.git
```

Go to the project directory

```bash
  cd gutiermoran.com
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
